# Evidence Packs

This directory is optional during Phase 1.

Use it only when a profile needs more support than normal inline numbered citations and a `References` section can provide cleanly.

Examples:

- multiple disputed source dates that need side-by-side comparison
- scanned historical tables or screenshots that need preservation
- claim-level evidence extracts that would make the main profile unreadable

Do not move ordinary citations here by default. Keep normal evidence inside the profile unless adjudication becomes noisy.
